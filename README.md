# 🚀 Sorting Algorithm Visualizer

An interactive and visually engaging **sorting algorithm visualizer** built with **React, Tailwind CSS, and Framer Motion**. This project demonstrates how popular sorting algorithms work step-by-step using smooth animations and intuitive UI controls.

## 🧠 How It Works

### Quick Sort

* Selects a **pivot element**
* Partitions the array into:

  * Elements smaller than pivot
  * Elements greater than pivot
* Recursively sorts subarrays

### Bubble Sort

* Repeatedly compares adjacent elements
* Swaps them if they are in the wrong order
* Largest elements "bubble" to the top
